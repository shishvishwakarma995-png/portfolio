🚀 Developer Portfolio

A modern, fully responsive developer portfolio built with React and Framer Motion to showcase my projects, technical skills, and contact information.

🔗 Live Demo

👉 [https://portfolio-iota-weld-99.vercel.app/]

✨ Features

Responsive design (mobile, tablet, desktop)

Modern UI with Tailwind CSS utility styling

Smooth animations using Framer Motion

Projects showcase section

Skills section with tech stack

Contact section with email copy-to-clipboard

GitHub profile integration

Clean and minimal layout

🛠️ Tech Stack

Frontend: React, JavaScript, HTML5

Styling: Tailwind CSS

Animations: Framer Motion

Build Tool: Vite

Version Control: Git & GitHub

⚙️ Installation & Setup

Clone the repository:

git clone https://github.com/shishvishwakarma995-png/portfolio.git
cd portfolio

Install dependencies:

npm install

Run the development server:

npm run dev

Build for production:

npm run build
